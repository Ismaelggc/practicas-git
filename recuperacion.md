## Recuperación del archivo errores.sh  
Se ha borrado el archivo errores.sh por error mientras se trabajaba por lo que se ha tenido que restaurar usando un commit anterior.  
Para ello hemos usado el comando `git log` para seleccionar el commit anterior al error. Desde aquí hemos usado el comando `git reset` para volver al estsado anterior.  
Aquí nos avisa de que un cambio en los archivos y mediante el comando `git restore` recuperamos el archivo.
